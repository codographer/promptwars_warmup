# Cooking To-Do List Planner Micro-App

A structured meal planning flow generated dynamically by AI based on your day's schedule, diet preferences, and target budget. Built on Next.js, Tailwind CSS, and the Google Gemini API.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Configure Environment Variables:**
   Create a `.env.local` file in the root directory:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to test.
